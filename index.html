Pines Tags jQuery Plugin

Pines Tags is a JavaScript content tag editor plugin for the jQuery JavaScript library, developed by Hunter Perrin as part of Pines. It is licensed under the GNU Affero GPL. It is designed to provide maximum flexibility, while still being easy to use and easy to implement. The original input element is hidden and a clone input element is created to allow the user to edit and create tags. This new input is exactly the same as the original, except "__ptags" is appended to the name and ID. The original input's value is kept current with all the tags which are entered. Try out different themes using the selector in the top right corner of this page.

This is the development site. It is a showcase of bleeding edge features and code that may work only partially or not at all. If you are looking for a stable version, please visit the main site.

This is the stable site. You can download code here that's tested and proven to work. However, if you like to live dangerously, you can also check out the development site.

Features

  • Preserves original input element. The value is updated, so you don't have to alter your scripts.
  • Tags can be edited by clicking them.
  • Supports any custom delimeters.
  • XHTML 1.0 Strict compliance.

Installation

Installation is simple. Include jQuery and jQuery UI (only the CSS is required). Then include the following files:

  • jquery.ptags.js or jquery.ptags.min.js (Minified)
  • jquery.ptags.default.css
